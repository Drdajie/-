【 赛迪网 讯 】 5 月 1 日 消息 ， Sun 新任 首席 执行官 乔纳森 ・ 施瓦茨 ( Jonathan Schwartz ) 上任 的 第一个 100 天 将要 出现 一些 有趣 的 事情 。 首先 ， 他 要 主持 一个 公司 内部 有关 是否 应该 开放 Java 源代码 的 争论 。 据 eWEEK . com 网站 报道 ， Sun 内部 的 消息灵通 人士 透露 ， Sun 将 在 今年 5 月 16 日 举行 的 JavaOne 会议 上 展开 有关 是否 开放 Java 源代码 的 争论 。 eWEEK . com 网站 没有 找到 施瓦茨 对此 消息 发表 评论 。 然而 ， 一位 不 愿意 透露 姓名 的 接近 Sun 公司 的 消息灵通 人士 称 ， 由于 施瓦茨 现在 担任 了 首席 执行官 ， 支持 开放 Java 源代码 的 人们 正在 努力 与 他 沟通 以便 在 JavaOne 会议 召开 之前 把 这个 观点 反映 到 他 那里 。 人们 推测 出 ， 在 这次 JavaOne 会议 上将 会 宣布 许多 开放 Java 源代码 的 消息 。 施瓦茨 最终 会 对 Java 采取 什么 决策 还有 待 观察 。 不过 ， 这是 施瓦茨 要 做 的 许多 事情 之一 。 4 月 24 日 刚刚 接替 斯科特 ・ 麦克尼利 担任 Sun 首席 执行官 的 施瓦茨 要 使 华尔街 满意 并且 要 对 Sun 实施 重组 以便 保证 该 公司 的 持续 盈利 。 Sun 自从 2001 年 以来 的 年报 一直 亏损 ， 在 截止 到 2006 年 3 月 的 这个 财年 亏损 了 2.17 亿美元 。 与此同时 ， 有 一些 人 对 施瓦茨 持 怀疑 态度 。 金融服务 公司 摩根 大通 在 4 月 25 日 发表 的 研究 报告 称 ， 它 担心 施瓦茨 给 Sun 带来 的 变化 也许 比 外部 人员 带来 的 变化 还要 少 。 但是 ， 施瓦茨 自己 充满 了 信心 。 他 说 ， 首先 ， 我们 处在 一个 增长 的 行业 。 在 我们 生命 的 其余 时间 里 ， 网络 行业 将 一直 在 增长 ， 它 将 需要 Sun 制造 的 产品 。 Sun 有 优越 的 地位 利用网络 行业 增长 的 商机 。 Sun 到 目前为止 仍 反对 开放 Java 源代码 的 要求 。 主要 原因 是 Sun 担心 开放 Java 源代码 之后 会为 竞争对手 获取 和 改变 Java 敞开大门 ， 导致 Java 内核 出现 多种 分支 和 兼容 新 问题 。 2003 年前 担任 Sun 负责 网络 业务 的 首席 工程师 、 现任 旧金山 的 ActiveGrid 公司 的 首席 执行官 Peter Yared 赞成 开放 Java 源代码 。 他 说 ， 目前 最大 的 问题 是 如何 让 Java 使 Sun 的 股东 收益 ， 特别 是 在 Sun 从来 没有 用 Java 赚过 钱 的 情况 下 更 应该 如此 。 eWEEK 网站 听说 Sun 内部 正在 讨论 有关 修改 Java 许可证 条款 的 可能性 。 Sun 与 战略 合作伙伴 围绕 Java 以及 与 Java 相关 的 产品 和 服务 的 谈判 正在 进行 之中 。 作为 Sun 的 首席 执行官 ， 施瓦茨 现在 可以 对 开放 Java 源代码 的 呼吁 做出 反应 。 但是 ， 消息灵通 人士 称 ， Sun 内部 担心 的 是 开放 Java 源代码 可能 会 使 微软 和 IBM 在 市场营销 方面 超过 Sun 。
