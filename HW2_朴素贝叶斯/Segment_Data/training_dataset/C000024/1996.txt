英国 《 简氏 国际 防务 评论 》 2006 年 1 月 1 日 报道 日本航空自卫队 ( JASDF ) 搜索 和 救援飞机 U - 125A ( 霍克 800 ) 装备 AN / APS - 134LW 雷达 。 根据 日本 东芝 和 美国雷声公司 空间 和 机载 系统部 ( SAS ) 签订 的 860 万美元 的 合同条款 ， 向 日本航空自卫队 提供 最新 的 雷达 硬件 。 根据上述 合同 ， 雷声 公司 空间 和 机载 系统 部将 向 东芝公司 提供 7 部 接收机 - 激励 器 - 同步器 处理机 和 4 部 天线 支座 ， 随后 向 日本航空自卫队 交付 。 由 雷声 公司 和 东芝公司 合作 生产 AN / APS - 134LW 雷达 ( 日本自卫队 命名 为 APQ - 2 ) 。 东芝公司 为 APQ - 2 雷达 提供 发射机 和 天线 反射器 。 雷声 公司 向 东芝公司 供应 备件 ， 提供 维修 和 技术支持 。 AN / APS - 134LW 是 雷声 公司 SeaVue 海上 和 濒海 地区 监视 任务 系统 的 组成部分 。 它 将 提供 高性能 的 空中 导航 、 气象 探测 、 远距离 小 目标 搜索 和 探测 ， 这是 日本航空自卫队 执行 搜索 和 救援 任务 所 要求 的 。 目前 在 日本航空自卫队 共有 25 架 霍克 U - 125A 飞机 服役 ( 需求量 是 27 架 ) ， 飞机 的 日本 主 承包商 是 金松 ( Kanematsu ) 公司 。 东芝公司 的 雷达 APQ - 2 安装 在 机腹 下 的 雷达 罩 内 ， 可以 实现 360o 全向 扫描 。 一台 三菱 电机 制造 的 红外 成像 器装 在 机身 下方 的 一个 可收回 的 容器 内 。 飞机 还 将 携带 灯光 / 浮标 ， 机身 两侧 还有 观察 用 舷窗 。 该机 于 1995 年 开始 服役 。 根据 简氏 年鉴 ( 电子 任务 飞机 ) ， AN / APS - 134 / SeaVue 雷达 系列 除 装备 日本自卫队 外 ， 还有 澳大利亚 ( 装在 DASH - 8 和 Reims406 飞机 上 ) 、 意大利 ( 装在 ATR42 飞机 上 ) 、 墨西哥 ( 装在 C - 212 和 EMB - 145MP 飞机 上 ) 、 英国 ( 装在 " 猎迷 " R1 飞机 上 ) 以及 美国 ( P - 3 ) 使用 。
